TICKER OPTION {TICKER}

The ticker is available on the ticker list page

TIME FRAME OPTION {TIMEFRAME}

Daily => “1day”

Each 1 hour => “1hour”

Each 30 minutes => “30min”

Each 5 minutes => “5min”

Each 1 minutes => “1min”
